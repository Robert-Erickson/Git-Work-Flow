Learning how to use Git.

Git is the version control software. GitHub (https://education.github.com) and GitLab (https://github.uvm.edu) is a cloud service based on the version control software called Git. There are minor difference between the two cloud services. The unix commands are exaclty the same. A goole search can clear up any other issue you may have.

Starting https://education.github.comworking on your own.

Moving to https://education.github.comworking in a group or using branches.

Git commands

git remote add origin git@github.com:Robert-Erickson/Git-Work-Flow.git
git push -u origin master

on yoru local machine
cd Desktop/folder-whatever-you-called-it
